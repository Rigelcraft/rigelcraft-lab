VERSIONES DE LA WEB

Bitácora de desarrollo de Rigelcraft Studio

En esta sección documentamos la evolución de nuestra plataforma. Desde los primeros días en WordPress hasta la optimización total con código puro y hasta las últimas versiones.

Versión: 2.1.0 – Optimización completa para dispositivos móviles y apertura del sistema de historial de versiones.

V: 2.1.0 - La Actualización de Movilidad

Optimización completa para dispositivos móviles y apertura del sistema de historial de versiones.

  • 📱 Adaptabilidad: La interfaz ahora se ajusta automáticamente a celulares y tablets.
  • 📜 Historial Abierto: Se añade esta nueva sección para que la comunidad siga el progreso técnico de la web.
  • 🚀 Rendimiento: Eliminación de residuos de código para una carga más veloz.

V: 2.0.0 - El Salto al Código Puro

La gran migración técnica.

  • 💻 Adiós WordPress: Reconstrucción total de la web desde cero usando únicamente HTML5 y CSS3.
  • 🎨 Interfaz Personalizada: Control absoluto sobre el diseño, permitiendo efectos visuales imposibles en la versión anterior.
  • ⚡ Optimización Estructural: Limpieza de código para eliminar la pesadez de los plugins y mejorar la estabilidad.

V: 1.0.0 - El Inicio en WordPress

El origen de la plataforma.

  • 🌐 Lanzamiento Oficial: Primera versión de Rigelcraft Studio disponible para el público.
  • 📂 Contenido Base: Implementación inicial de las secciones de Galería, Guiones y Preestrenos.
  • 🛠️ Gestión CMS: Uso de herramientas de WordPress para establecer la estructura inicial del sitio.
